Uses

This is a living document of the software, tools, and hardware I use — maybe you find it useful 😊

Editor & Terminal

  • I primarily use my desktop PC with Windows 10 and occasionally my MacBook Pro with MacOS
  • Visual Studio Code
  • JetBrains Mono as preferred font
  • zsh + oh-my-zsh
    • spaceship as zsh prompt
    • Plugins: git, zsh-autosuggestions, virtualenv, yarn, zsh-syntax-highlighting, history, gh
  • Windows Terminal & iTerm2
  • nvm as Node.js version manager
  • On Windows 10 I use WSL 2 (Ubuntu) as package manager for developing, on MacOS I use MacPorts

Software

  • Adobe Photoshop & CaptureOne for photo editing
  • Cinema4D for 3D art + Arnold as renderer
  • Figma for designs
  • Duplicati for offline + online backup strategy
  • Enpass as password manager
  • f.lux for proper coding night sessions
  • FastStone Photo Resizer as image batch processing tool
  • ShareX for taking screenshots, recording quick videos/gifs
  • Streamlabs OBS for recording videos

Hardware

  • My main workstation has an AMD Ryzen 9 3900X, 32GB DDR4 G.Skill Flare X RAM, Palit GeForce RTX 2080 Super WGRP and a 1TB M.2 Corsair Force Series MP600 in a sleek Fractal Design Define R6 case
  • 2x Dell Ultrasharp U2515H
  • Corsair K70 LUX RGB as keyboard
  • Logitech Pro Wireless as mouse
  • Video Setup: Logitech C920 + 2x Neewer 5600k USB-LED lights (powered by UGREEN 36W USB Charger) + 2x SMALLRIG Mini Magic Arm
  • Audio Setup: SE Electronics V7 + Focusrite Scarlett Solo 3rd Gen + TritonAudio FetHead + Rode PSA1
  • FujiFilm X-T100 with those Fujinon lenses:
    • XF16-80mm F4 R OIS WR
    • XF16mm F2.8 R WR
